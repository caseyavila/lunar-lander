# lunar-lander
Simple OpenGL game project

## Some Features
- Apollo 11 lander with a rocket booster
- Newtonian physics
- Simple/clever collision detection
- Metallic shader!

## For the (far) Future
- Framerate-independent physics
- An actual loading screen
- Better loading time
- An actual ending screen
